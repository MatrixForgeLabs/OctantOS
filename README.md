# OctantOS Public

This repository is the public information hub for OctantOS.

## Scope

- Public product updates
- High-level roadmap and changelog
- Security disclosure process
- Community discussions and issue tracking

## Source Code

The OctantOS source code is not hosted in this repository.

This repository intentionally excludes implementation code, infrastructure internals, and operational details.

## Contents

- `ROADMAP.md`
- `CHANGELOG.md`
- `FAQ.md`
- `SECURITY.md`
- `CONTRIBUTING.md`

## Contact

For security reports, follow `SECURITY.md`.
For general questions, open a Discussion or Issue.
